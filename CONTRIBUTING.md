# Contributing to django-netcast

Thanks for your interest! Here's how to contribute:

## Getting Started
1. Fork the repo
2. Clone your fork: `git clone https://github.com/<your-username>/django-netcast.git`
3. Install dev dependencies: `pip install -e ".[dev]"` or `pip install pytest pytest-django`
4. Run tests: `pytest`

## Pull Requests
- Create a feature branch: `git checkout -b feature/your-feature`
- Write tests for new functionality
- Ensure all tests pass before submitting
- Open a PR against `main`

## Reporting Bugs
- Use [GitHub Issues](https://github.com/sagar9187/django-netcast/issues)
- Include Python version, Django version, and OS
- Include steps to reproduce

## Code Style
- Follow PEP 8
- Keep functions small and focused
